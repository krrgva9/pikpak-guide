有，但额度相当大——每月云下载流量 40TB、下行流量 4TB、上传流量 1TB。正常个人用户基本不可能触碰到这个上限，只有批量刷流量的异常账号才会被限制。

**Q：iOS 用户怎么添加磁力链接？**
